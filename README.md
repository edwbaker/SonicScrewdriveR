# Sonic ScrewdriveR
This package provides basic tools for manipulating sound files for analysis. 

## Installation

### CRAN
````R
install.packages("sonicscrewdriver")
library(sonicscrewdriver)
````

### Latest master
````R
install.packages("devtools")
library(devtools)
install_github("edwbaker/SonicScrewdriveR")
library(sonicscrewdriver)
````
