# Helper function to convert a Wave-like object to scikit-maad spectrogram

Helper function to convert a Wave-like object to scikit-maad spectrogram

## Usage

``` r
.spectrogram_maad_power(object)
```

## Arguments

- object:

  A Wave or WaveMC object (or Tagged equivalent), or a spectrogram_maad
  object

## Value

A spectrogram_maad object
