# Zero spectrum

This function takes a spectrum from seewave and creates a new
zero-valued spectrum with the same structure.

## Usage

``` r
zeroSpectrum(s1)
```

## Arguments

- s1:

  Spectrum to emulate the structure of.

## Value

A zero-valued spectrum.

## Examples

``` r
if (FALSE) { # \dontrun{
zeroSpectrum(spec)
} # }
```
