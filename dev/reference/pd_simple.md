# Simplified pulse detection using Dietrich (2004)

Detects pulses in a Wave.

## Usage

``` r
pd_simple(
  wave,
  U = 120,
  gamma = 0.05,
  alpha = 1.4,
  scaling = 32,
  V = 480,
  psi = 1
)
```

## Arguments

- wave:

  A Wave object

- U:

  Window length

- gamma:

  Gamma

- alpha:

  Alpha

- scaling:

  Scaling

- V:

  V Window length

- psi:

  Psi
