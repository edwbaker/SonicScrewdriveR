# An S4 class to represent a spectrogram from scikit-maad.

An S4 class to represent a spectrogram from scikit-maad.

## Slots

- `Sxx`:

  The spectrogram matrix

- `tn`:

  The time vector

- `fn`:

  The frequency vector

- `extents`:

  The extents of the spectrogram. A list of tmin, tmax, fmin, fmax.
