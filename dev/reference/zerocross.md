# Identify zero crossings in a Wave object

Returns a vector of the position (in samples) of zero crossings in a
Wave object

## Usage

``` r
zerocross(wave)
```

## Arguments

- wave:

  A Wave object

## Value

A vector of zero crossing locations

## Examples

``` r
if (FALSE) { # \dontrun{
zerocross(sheep)
} # }
```
